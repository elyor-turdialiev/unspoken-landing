Política de privacidad

Tus palabras son tuyas.

Esta página explica qué datos recopila Unspoken, cómo los protegemos y qué nunca haremos con ellos. Lo escribimos para personas, no para abogados.

Última actualización: 29 de abril de 2026

Quiénes somos

Unspoken es operado por Elyorbek Turdialiev, con sede en Riga, Letonia. A efectos del Reglamento General de Protección de Datos de la UE (RGPD), Elyorbek Turdialiev es el responsable del tratamiento.

Contacto: hello@unspoken.app

Qué recopilamos

Información de cuenta: Tu dirección de correo y una contraseña cifrada. Los necesitamos para crear y proteger tu cuenta. No pedimos nombre, teléfono, ubicación ni otra información identificativa.

Tus escritos: Las entradas que escribes se almacenan en nuestra base de datos para que puedas acceder a ellas y para generar tus reflexiones semanales. Se cifran en tránsito (HTTPS) y se almacenan en servidores de Supabase (en la UE).

Selección de estado de ánimo: Cuando etiquetas una entrada con un estado de ánimo (ej. "tranquilo", "cansado"), esa selección se almacena junto a tu entrada.

Ajustes: Tu zona horaria, preferencia de idioma y otros ajustes que configures.

Datos de reflexiones: Las reflexiones semanales generadas a partir de tus entradas se almacenan para que puedas revisarlas.

Datos técnicos: Recopilamos datos técnicos mínimos: tipo de navegador, marcas de tiempo aproximadas. No usamos rastreo analítico, píxeles publicitarios ni huellas digitales.

Cómo usamos tus datos

Para proporcionar el servicio: Tus entradas y datos de ánimo se usan para generar reflexiones semanales. Este procesamiento lo realiza la API de Anthropic (Claude). Tus entradas se envían a Anthropic exclusivamente para generar tu reflexión. La política de Anthropic establece que los datos enviados por API no se usan para entrenar sus modelos.

Para comunicarnos contigo: Podemos enviarte tu reflexión semanal por correo (si lo has elegido). No enviamos correos de marketing.

Para mejorar el servicio: Podemos analizar patrones de uso agregados y anónimos. Nunca analizamos entradas individuales para este fin.

Lo que nunca haremos

Nunca venderemos tus datos a anunciantes ni a terceros.

Nunca te mostraremos publicidad.

Nunca leeremos tus entradas manualmente a menos que tú lo solicites explícitamente.

Nunca compartiremos tus entradas con otros usuarios a menos que tú elijas hacerlo.

Nunca usaremos tus entradas para entrenar modelos de IA. La API de Anthropic procesa tus entradas solo para generar tu reflexión y las descarta después.

Servicios de terceros

Supabase: Alojamiento de base de datos y autenticación. Servidores en la UE.

Anthropic (Claude API): Procesa tus entradas para generar reflexiones semanales. Los datos no se retienen ni se usan para entrenamiento.

Netlify: Aloja el sitio web de marketing (no se procesan datos de usuario).

Resend: Servicio de entrega de correo para enviar reflexiones. (Configuración pendiente.)

Retención de datos

Tus entradas y reflexiones se almacenan mientras tu cuenta exista. Puedes eliminar entradas individuales en cualquier momento. Puedes eliminar tu cuenta y todos los datos en Ajustes. La eliminación es permanente e irreversible.

Puedes exportar todos tus datos como archivo JSON en cualquier momento.

Tus derechos bajo el RGPD

Si estás en el Espacio Económico Europeo, tienes derecho a: acceder a tus datos personales; corregir datos inexactos; solicitar la eliminación; exportar datos; oponerte al procesamiento; y retirar el consentimiento en cualquier momento.

Para ejercer estos derechos, escríbenos a hello@unspoken.app. Responderemos en 30 días.

Privacidad de menores

Unspoken está disponible para usuarios de 13 años o más. Los usuarios de 13 a 15 años necesitan consentimiento de un padre o tutor. Los usuarios de 16 años o más pueden consentir de forma independiente. Si descubrimos que hemos recopilado datos de un menor de 13 sin consentimiento, los eliminaremos.

Si eres padre o tutor y crees que tu hijo creó una cuenta sin tu consentimiento, contáctanos: hello@unspoken.app.

Cookies

Unspoken usa solo cookies esenciales para autenticación y gestión de sesiones. No usamos cookies de rastreo, analíticas ni publicitarias. No hay nada que rechazar porque no te rastreamos.

Seguridad

Todos los datos se transmiten por HTTPS. Las contraseñas se cifran y nunca se almacenan en texto plano. Seguimos prácticas de seguridad estándar de la industria.

A pesar de nuestros esfuerzos, ningún sistema es perfectamente seguro. Si descubres una vulnerabilidad, contáctanos: hello@unspoken.app.

Cambios en esta política

Si hacemos cambios materiales, te notificaremos por correo o dentro de la app. La fecha de actualización refleja la última revisión.

Contacto

Para cualquier pregunta sobre esta política, escríbenos a hello@unspoken.app.
